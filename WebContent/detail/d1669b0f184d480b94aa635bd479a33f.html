    Base64编码简介

    编码方法，不能用于加密，将二进制转换为可显示的字符。

    Base64编码

    base64一共64个索引（0-63），而63转换为二进制为111111，64个字符最多使用6个bit即可表示，而一个字节8个bit。
那么如何使用6个bit表示8个bit?
base64将3个传统字节转化为4个base64字符表示。

    Java示例

    jar：commons-codec-1.9.jar

    import org.apache.commons.codec.binary.Base64;

    public String encode(String str) {
	byte[] encodeBytes = Base64.encodeBase64(str.getBytes());
	return new String(encodeBytes);
}

public String decode(String str) {
	byte[] decodeBytes = Base64.encodeBase64(str.getBytes());
	return new String(decodeBytes);
}
