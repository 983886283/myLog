    Varnish安装

    Varnish依赖包
yum install gcc
yum install pcre*
yum install libedit-dev*
下载Varnish使用wget命令
yum install wget
下载Varnish安装包
wget -c http://varnish-cache.org/_downloads/varnish-6.0.0.tgz
解压
tar zvxf varnish-6.0.0.tgz
进入varnish-6.0.0进行配置
cd varnish-6.0.0/
设置路径
./configure --prefix=/usr/common/varnish
编译安装
make
make install
