    使用cmd命令执行Java中的main方法

    结构：java -cp jar路径 main方法路径

    注意：结构中的空格不要漏，main方法路径为class文件路径并非java文件路径

    1.bat脚本（非必需）

    cd C:\c\song\bin
java -cp ..\resources\lib\*; com.cjp.threads.Test canshu1 canshu2
pause

    2.main方法传参，接受参数

    	public static void main(String[] args)  {		
		if(args!=null&&args.length>0){
			for (int i = 0; i < args.length; i++) {
				System.out.println(args[i]);
			}
		}
	}
